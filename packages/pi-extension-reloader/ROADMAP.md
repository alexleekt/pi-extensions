# Roadmap

## Ideas

- [ ] Watch mode: `/rebuild-extension --watch <name>` to auto-rebuild on file changes
- [ ] Batch rebuild: `/rebuild-extension` with no args to rebuild all local extensions
- [ ] Dry-run flag: `/rebuild-extension --dry-run <name>` to preview what would change
- [ ] Cache size report: show total jiti cache size and per-extension breakdown
- [ ] Post-rebuild health check: verify the extension loads without errors after reload
